# Brute Force Lab

A hands-on cybersecurity simulation that demonstrates how brute force 
password attacks work in practice.

Built with Python (Flask) and vanilla JavaScript.

## What it does
- Takes any password up to 5 characters
- Simulates a real brute force attack attempt by attempt
- Shows total attempts, time taken, and password strength rating
- Logs every result automatically

## How to run it locally
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Run: python app.py
4. Open your browser at http://localhost:5000

## Tech used
- Python & Flask
- HTML, CSS, JavaScript
